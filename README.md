# ☕ ShrubBrewWorld – Coffee Website

A modern, animated coffee website with a **cinematic intro (coffee brewing video)** and smooth UI experience. Built using HTML, CSS, and JavaScript.

---

## 🚀 Features

* 🎬 **Coffee Brewing Intro Video**

  * Plays before entering homepage
  * Smooth transition to main site

* 🎨 **Modern UI Design**

  * Dark premium coffee theme
  * Responsive layout

* ✨ **Animations**

  * Scroll-based animations
  * Hover effects
  * Smooth transitions

* ☕ **Coffee Menu Section**

  * Espresso, Latte, Cappuccino, etc.
  * Interactive UI cards

* 🖼️ **Gallery with Lightbox**

  * Click images to view in popup
  * Navigation between images

* 📅 **Reservation System**

  * Book table form
  * User-friendly interface

* 📱 **Responsive Design**

  * Works on mobile, tablet, desktop

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Animations + Responsive Design(both web as well as mobile interface usable))
* JavaScript (DOM + Interactivity)

---

## 🎬 How It Works

1. User opens website
2. Coffee brewing intro video plays ☕
3. After video ends → Homepage loads
4. User explores menu, gallery, and booking
